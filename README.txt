ANCORA WEBSITE — FIRST BUILD

WHAT IS INCLUDED
- Responsive one-page marketing website
- Actual Ancora screenshots only
- macOS and Windows positioning
- Maintenance, defects, inventory, tasks, certificates and voyage records
- Early-access $99/month section
- Draft privacy and terms pages

BEFORE PUBLISHING
1. Replace every occurrence of hello@yourdomain.com in index.html.
2. Confirm whether you want the $99/month price public.
3. Replace the draft Privacy Policy and Terms with professionally reviewed documents.
4. Add your final domain, company name and legal contact details.
5. Test all screenshots for vessel-sensitive information before public launch.

GODADDY UPLOAD — CPANEL / WEB HOSTING
1. Open GoDaddy > My Products > Web Hosting > Manage.
2. Open cPanel Admin, then File Manager.
3. Open public_html.
4. Remove or rename the existing default index file.
5. Upload all files and the assets folder from this package into public_html.
6. Confirm index.html is directly inside public_html, not inside another folder.
7. Visit the domain and clear browser cache if an old page appears.
8. Enable the free SSL certificate and force HTTPS in GoDaddy.

IF YOU ONLY HAVE GODADDY WEBSITE BUILDER
Website Builder does not allow this full custom static site to be uploaded. You would need a Web Hosting/cPanel plan, or host the site elsewhere (for example Cloudflare Pages, Netlify or Vercel) and point the GoDaddy domain to it.
